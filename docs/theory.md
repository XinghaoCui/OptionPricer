# 金融理论背景

本文档简要介绍期权定价的金融理论基础，作为算法实现的"为什么"补充。

---

## 1. 什么是期权？

**期权（Option）** 是一种衍生品合约，其持有人**有权（而非义务）**在约定时间以约定价格买入或卖出标的资产：

* **看涨期权（Call）**：有权按行权价 K 买入标的；
* **看跌期权（Put）**：有权按行权价 K 卖出标的；
* **欧式期权（European）**：只能在到期日 T 行权；
* **美式期权（American）**：到期日 T 之前的任意时刻都可以行权。

**到期收益（payoff）**：

```
Call payoff = max(S_T - K, 0)
Put  payoff = max(K - S_T, 0)
```

---

## 2. Black-Scholes 模型假设

1. 标的资产价格服从几何布朗运动（GBM）：
   ```
   dS = (r - q) S dt + σ S dW
   ```
   其中 r 为无风险利率，q 为连续股息率，σ 为波动率，W 是标准布朗运动。

2. 无套利、无交易成本、可任意分割、可无成本卖空、利率与波动率为常数。

3. 在风险中性测度下贴现期望即为期权价格：
   ```
   V_0 = e^(-rT) · E^Q[ payoff(S_T) ]
   ```

---

## 3. Black-Scholes 公式

欧式 Call：

```
C = S e^(-qT) N(d₁) - K e^(-rT) N(d₂)
```

欧式 Put：

```
P = K e^(-rT) N(-d₂) - S e^(-qT) N(-d₁)
```

其中：

```
d₁ = [ln(S/K) + (r - q + σ²/2) T] / (σ√T)
d₂ = d₁ - σ√T
N(·) 是标准正态分布的 CDF
```

---

## 4. Put-Call Parity（看涨看跌平价关系）

由无套利原理可推出：

```
C - P = S·e^(-qT) - K·e^(-rT)
```

这是所有欧式期权定价器都必须满足的恒等式，本项目在 `tests/test_black_scholes.py` 中做了严格验证。

---

## 5. 为什么美式期权没有解析解？

美式期权可以在任意时刻行权，本质是**最优停时（Optimal Stopping）问题**：

```
V_0 = sup_{τ ≤ T}  E^Q[ e^(-rτ) · payoff(S_τ) ]
```

其中 τ 是一个停时（stopping time）。这等价于一个自由边界问题（free boundary problem）的偏微分方程，**一般无闭式解**，必须靠数值方法：

* **二叉树 / 三叉树**：在每个节点比较"立即行权"与"继续持有"——**本项目方法**；
* **有限差分法**（解 PDE）；
* **Longstaff-Schwartz 回归 MC**（高维问题首选）。

---

## 6. Merton 定理：无股息时美式 Call = 欧式 Call

**定理**：当标的资产不支付股息时（q = 0），美式 Call 与欧式 Call 价格相等——即**提前行权永远不优**。

**直觉证明**：

* 若你提前行权得到 S_t - K，等于放弃了 max(S_T - K, 0) 与 S_t - K 之间的差额（即"时间价值"）；
* 同时还失去了利率 r 在剩余时间 (T-t) 内为 K 的"贴现收益"；
* 不行权而直接卖出期权得到的市场价 ≥ S_t - K（因为期权有时间价值），所以**总是更优**。

本项目通过 200 步二叉树验证了这一定理（误差 = 0.00e+00），见 `examples/pricing_demo.py`。

## 7. Greeks 的金融含义与实践用途

| Greek | 数学定义 | 金融含义 | 实务用途 |
| --- | --- | --- | --- |
| Delta | ∂V/∂S | 标的价变化 1 元，期权价变化多少 | **Delta 对冲**：做市商通过买 -Δ 份标的对冲 |
| Gamma | ∂²V/∂S² | Delta 的变化率（凸性） | **Gamma 风险**：大幅波动下 Delta 对冲失效 |
| Vega | ∂V/∂σ | 波动率变化 1% 期权价变化多少 | **波动率交易**：σ 上升时多头 Vega 获利 |
| Theta | ∂V/∂T | 时间衰减（持有期权每天损失多少） | **卖方策略**：写期权赚时间价值 |
| Rho | ∂V/∂r | 利率变化 1% 期权价变化多少 | 利率敏感型组合（长期权） |

## 8. 隐含波动率（IV）：市场的"波动率预期"

BS 公式有 5 个输入参数：S、K、T、r、σ。其中只有 σ 不能直接观察——它是市场对**未来**波动率的预期。

实务做法：给定市场期权价 V*，反解满足 BS(σ) = V* 的 σ，称为**隐含波动率（Implied Volatility）**。

IV 在量化金融中无处不在：

* **波动率曲面**（implied vol surface）：把不同 K 和 T 的 IV 排列成曲面，是衍生品交易的核心可视化；
* **波动率微笑 / 倾斜**：实际 IV 在不同 K 下不是常数，呈"微笑"形态，反映 BS 假设的偏差；
* **VIX 指数**：S&P 500 期权的加权 IV，被称为"恐慌指数"。

本项目实现了基于 BS 的 IV 求解器（`src/implied_vol.py`），可作为后续构建波动率曲面的基础工具。

## 9. 蒙特卡洛在金融中的价值

虽然欧式期权用 BS 即可，但许多复杂衍生品**没有解析解**，蒙特卡洛是唯一可行的方法：

* **路径依赖期权**：亚式期权（payoff 依赖路径均值）、回望期权（依赖路径极值）；
* **多资产期权**：篮子期权、彩虹期权——高维问题中 MC 不受维度诅咒影响；
* **风险度量**：VaR、CVaR、压力测试——本质都是分位数估计；
* **信用衍生品**：CDO 损失分布。

本项目实现的"对偶变量 + 控制变量"是工业级 MC 系统的标准技术。
