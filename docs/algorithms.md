## 1. CRR 二叉树定价

### 1.1 模型设定

将到期时间 T 等分为 N 步，每步 Δt = T/N。假设每步标的资产价格：

* 以概率 p 上涨为原价的 u 倍
* 以概率 1−p 下跌为原价的 d 倍

为了与 GBM 模型的方差匹配，Cox-Ross-Rubinstein (1979) 取：

```
u = exp(σ√Δt)
d = 1/u
p = [exp((r-q)Δt) - d] / (u - d)    （风险中性概率）
```

### 1.2 重组树的关键性质

由于 `ud = du = 1`，第 n 步上涨 i 次的节点和下跌 j 次的节点（i = j 时）落在同一价格点上——这就是 **"重组性"（recombining）**。带来的好处：

* 第 n 步只有 **n+1 个不同节点**（而非 2ⁿ 个）；
* 用一维数组 `values[0..N]` 即可表示整层节点的价值；
* 倒推时可**原地更新**（in-place），无需额外空间。

### 1.3 算法伪代码

```text
procedure CRR_BINOMIAL_PRICE(option, N):
    dt ← T / N
    u ← exp(σ√dt);  d ← 1/u;  disc ← exp(-r·dt)
    p ← (exp((r-q)·dt) - d) / (u - d)

    # 1. 叶子层 payoff
    for i ← 0 to N:
        S_Ni ← S · u^i · d^(N-i)
        values[i] ← max(S_Ni - K, 0)   if Call else max(K - S_Ni, 0)

    # 2. 自底向上回溯（DP）
    for n ← N-1 down to 0:
        for i ← 0 to n:
            continuation ← disc · (p · values[i+1] + (1-p) · values[i])
            if option.style = AMERICAN:
                S_ni ← S · u^i · d^(n-i)
                intrinsic ← payoff(S_ni)
                values[i] ← max(intrinsic, continuation)
            else:
                values[i] ← continuation

    return values[0]
```

### 1.4 复杂度分析

* **时间**：内层循环总迭代次数 = ∑(n+1) for n=0..N-1 = N(N+1)/2 = **O(N²)**
* **空间**：仅维护 `values[0..N]`，**O(N)**

### 1.5 收敛性

CRR 价格关于步数 N 的收敛阶为 **O(1/N)**，且存在"奇偶振荡"现象——奇数步偏低、偶数步偏高（或反之），这是树定价的经典数值特征。详见 `docs/screenshots/convergence_binomial.png`。

## 2. Boyle 三叉树定价

### 2.1 模型设定

三叉树每步价格有三种走向：上涨为 u、不变（m=1）、下跌为 d=1/u。Boyle (1986) 的参数化：

```
u = exp(λσ√Δt),  d = 1/u,  m = 1
drift = r - q - σ²/2
p_u = 1/(2λ²) + drift·√Δt / (2λσ)
p_d = 1/(2λ²) - drift·√Δt / (2λσ)
p_m = 1 - p_u - p_d
```

其中 λ 是自由参数；Kamrad & Ritchken (1991) 证明 **λ = √3 时收敛性最好**。

### 2.2 重组性

`u·d = 1` 且 `m = 1`，所以第 n 步只有 **2n+1 个节点**，仍可用一维数组表示。

### 2.3 复杂度

* **时间**：∑(2n+1) for n=0..N-1 = N² → **O(N²)**
* **空间**：**O(N)**

### 2.4 三叉树 vs 二叉树

| 维度 | CRR 二叉树 | Boyle 三叉树 |
| --- | --- | --- |
| 每步节点数 | 2 | 3 |
| N 步后总节点 | N(N+1)/2 | N² + 1 |
| 收敛阶 | O(1/N) | O(1/N²)（特定参数下） |
| 数值稳定性 | 短期 / 高波动率下 p 可能越界 | 通过 λ 调节，更稳定 |

实测在相同步数下三叉树误差通常更小，但每步计算开销也更大。

## 3. 蒙特卡洛 + 方差缩减

### 3.1 普通蒙特卡洛

在风险中性测度下，标的资产到期价格服从 GBM：

```
S_T = S · exp[(r - q - σ²/2)T + σ√T · Z],   Z ~ N(0,1)
```

抽样 n 条独立路径，估计期权价格：

```
V̂ = e^(-rT) · (1/n) · Σᵢ payoff(S_T^(i))
```

标准误差：`SE = σ_payoff / √n`，即 **O(1/√n)** 收敛。

### 3.2 对偶变量法（Antithetic Variates）

每次抽样 Z 同时使用 −Z，构造一对样本：

```
X⁺ = payoff(S · exp(μT + σ√T · Z))
X⁻ = payoff(S · exp(μT - σ√T · Z))
X̄ = (X⁺ + X⁻) / 2
```

由于 X⁺ 与 X⁻ 负相关（当 payoff 关于 Z 单调时），有：

```
Var(X̄) = (Var(X⁺) + Var(X⁻) + 2·Cov(X⁺, X⁻)) / 4 < Var(X⁺) / 2
```

**额外好处**：只需生成 n/2 个随机数即可得到 n 个样本。

### 3.3 控制变量法（Control Variates）

引入一个已知期望的变量 Y，构造修正估计：

```
X' = X - β · (Y - E[Y])
```

当 β = Cov(X,Y)/Var(Y) 时，Var(X') 取最小值：

```
Var(X') = Var(X) · (1 - ρ²_{X,Y})
```

ρ 是 X 与 Y 的相关系数，**相关性越高方差缩减越大**。本项目用 Y = e^(-rT)·S_T，其期望 E[Y] = S·e^(-qT) 已知。对于欧式 Call/Put，X 与 Y 的相关系数 |ρ| 通常 > 0.95，方差可缩减 **10 倍以上**。

## 4. 隐含波动率求解

### 4.1 问题陈述

给定市场价 V*，反解满足 BS(σ) = V* 的 σ。这是一维方程求根问题。

### 4.2 牛顿迭代法

利用 Vega = ∂V/∂σ 作为导数：

```
σ_{n+1} = σ_n - (BS(σ_n) - V*) / Vega(σ_n)
```

**优点**：平方收敛（每步迭代误差由 |e_n|² 量级缩小）。
**缺点**：Vega 接近 0 时（深度虚值或深度实值期权）发散。

### 4.3 二分法兜底

对单调函数 f(σ) = BS(σ) - V*，已知 f 在区间端点异号时，反复取中点：

```
while hi - lo > tol:
    mid = (lo + hi) / 2
    if f(lo) · f(mid) < 0:  hi = mid
    else:                   lo = mid
```

**优点**：绝对稳定，每步区间长度减半（线性收敛 O(log₂(1/tol))）。
**缺点**：慢于牛顿法。

### 4.4 复合策略

本项目先尝试牛顿法（fast path），仅在以下情况切换二分法（safety net）：

1. Vega < 1e-10（导数太小）
2. 迭代值越界（σ ≤ 0 或 σ > 10）
3. 牛顿法迭代次数耗尽未收敛

这是"乐观快速路径 + 鲁棒兜底路径"模式的典型实现，在金融工程库中（QuantLib、py_vollib）广泛使用。

## 5. Greeks 的有限差分实现

### 5.1 中心差分

```
∂V/∂x ≈ [V(x+h) - V(x-h)] / (2h)        误差 O(h²)
∂²V/∂x² ≈ [V(x+h) - 2V(x) + V(x-h)] / h²    误差 O(h²)
```

中心差分比单侧差分 `[V(x+h) - V(x)] / h`（误差 O(h)）精度高一阶。

### 5.2 通用性

有限差分法**完全独立于定价器**——只需把 `pricer(option)` 作为黑盒调用三次（左、中、右），就能得到任意一阶 Greek。这使得本模块可用于：

* 欧式期权（验证 BS 解析 Greeks 的正确性）；
* **美式期权**（无解析式，必须靠差分）；
* **奇异期权**（亚式、回望式等，同样只能靠差分）。

详见 `examples/greeks_demo.py`，解析与差分的相对误差均在 1e-3 以下。
